# サンプル OpenHands - TODO CLI アプリケーション

これは Python で書かれたシンプルなコマンドライン TODO アプリケーションです。

## 機能

- TODOの追加
- TODOの一覧表示
- TODOの編集
- TODOの削除
- TODOの完了マーク機能

## 使用方法

以下のコマンドでアプリケーションを起動します:

    python3 todo_cli.py

このアプリケーションはインタラクティブな対話シェルとして動作します。  
入力プロンプトやステータスメッセージは、視認性向上のために緑色で表示されます。

## 注意事項

このアプリケーションはメモリ上のストレージを使用しています。  
アプリケーション終了時にすべての変更は失われます。

必要に応じて、検索、優先度設定、期限管理などの機能拡張を行ってください.
