# Simple TODO App\n\nこのプロジェクトは、GoのEchoフレームワークとTypeScriptで開発されたReactフロントエンドを使用したシンプルなTODOアプリの雛形です。\n\n## プロジェクト構成\n\n- **server**: Goサーバーコード（Echo使用）\n- **client**: Reactフロントエンドコード（TypeScript使用）\n- **docs**: OpenAPI仕様書（`/openapi.yaml` に配置）\n\n## 起動方法\n\n### サーバーの起動\n1. ターミナルで `server` ディレクトリに移動します:\n   \n   cd server\n\n2. Goコマンドでサーバーを起動します:\n   \n   go run main.go\n\n   サーバーは  で待ち受けます。\n\n### クライアントの確認\nサーバー起動後、ブラウザで  にアクセスすると、トップページが表示されます。\n\n### OpenAPI仕様書の確認\nOpenAPI仕様書は以下のURLで確認できます:\n\n   http://localhost:53805/openapi.yaml\n\n## 今後の予定\n- Reactフロントエンドの機能拡張\n- TODO機能の実装とAPIエンドポイントの追加\n- OpenAPI仕様の充実化
